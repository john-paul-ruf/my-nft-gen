# my-nft-gen
Software used to create my NFT artwork on [Rarible](https://rarible.com/JohnRuf).

Warning: might not work out of the box as no image files are provided.

To Run:
create the directory 'src/img/output'
Diable sig, summons, and backdrop effect by setting the effect chance to zero on the effect card
Install node, if you haven't a
Navigate to the root directory
From a terminal, run the command: 'node 'src/index.js'
Wait for output.  Can take 30 minutes to hours depending on effects, number of frames, and output image resolution
